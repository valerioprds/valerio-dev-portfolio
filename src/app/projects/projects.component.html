Product Admin App

Product Admin App is a mobile sales tracking app designed for efficient inventory management. It allows users to add, edit, and delete product entries with fields for name, price, and units sold, including image uploads from devices or camera captures. Built with Angular 17, Ionic 7, and Firebase 10.9, it offers seamless user registration and real-time updates. The app integrates Capacitor for native device features, ensuring a responsive and intuitive user experience. Perfect for businesses seeking streamlined product administration.

PeeMap

Discover PeeMap, the ultimate solution for urban dwellers and travelers alike! This innovative app effortlessly guides you to the nearest public toilets, ensuring you're never inconvenienced again. PeeMap stands out with its user-friendly interface and valuable features like real-time directions, cleanliness ratings, and the ability to add new toilet locations. PeeMap harnesses the power of Angular for a dynamic frontend, Mapbox GL for seamless map navigation, AngularFire for secure user authentication and data handling, and Material Design for an attractive, responsive user interface. We've integrated MongoDB for robust data management and Mapbox Directions to pinpoint the best routes to your destination.
